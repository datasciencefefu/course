Структура курса

Данное учебное пособие предназначено для студентов, обучающихся по программе высшего образования естественнонаучных и физико-технических направлений подготовки.

Для успешного освоения материала необходимо владение основами линейной алгебры и математического анализа, а также базовыми навыками программирования на языке Python.

Целями освоения курса является знакомство студентов с базовыми понятиями и методами анализа данных, примерами их использования в задачах обработки, анализа данных и информационного поиска, а также приобретение навыков аналитика данных и разработчика математических моделей, методов и алгоритмов анализа данных.

1 Основы статистического анализа В первой главе дается материал посвященный практическому применению математической статистики и теории вероятностей при анализе данных на Python. Для иллюстрации основных понятий и определений использованы возможности библиотек Matplotlib, Plotly, Pandas, Seaborn, NumPy, SciPy.
2 Предобработка данных Во второй главе приводится описание методов сбора и предобработки данных. Рассмотрены основные объекты библиотеки Pandas: DataFrame и Series. Большое внимание уделено приемам работы с регулярными выражениями с использованием библиотеки re, парсингу страниц (библиотеки requests для отправления HTTP-запросов и BeautifulSoup для преобразования HTML-файла в древовидную структуру) и методам обработки данных в формате JSON. Описаны способы работы с пропусками в датасетах, в том числе удаление дубликатов и объединение таблиц.
3 Визуализация данных В третьей главе рассматриваются основные методы визуализации данных с использованием библиотек Pandas, Matplotlib, Seaborn и Plotly.

Описание лабораторных работ и проектных заданий

В качестве закрепления изученного теоретического материала студентам предлагается выполнить ряд практических заданий. Практические материалы и задания для студентов выполнены с использованием Google Colaboratory - бесплатной интерактивной облачной среды для работы с кодом от Google. Основной акцент в практикуме сделан на проектных заданиях, которые позволяют студентам решать задачи, наиболее часто встречающиеся в реальной работе дата-сайентистов. Студенты осваивают основные приемы сбора и обработки данных, их статистического и исследовательского анализа, используя возможности библиотек Matplotlib, Plotly, Pandas, Seaborn, NumPy, SciPy и др.

  1. Проектное задание "AppStore"
    Для анализа используется набор данных о 7000 популярных мобильных приложениях в магазине приложений Apple в 2017 году. Приведено подробное описание содержимого датасета. Студентам предлагается на практике использовать различные виды статистических характеристик, индексирование срезами и логическую индексацию. Рекомендуется использовать преимущественно инструменты библиотеки Pandas, допустимо также использование NumPy.

  2. Лабораторная работа "Чтение и запись в файлы"
    Работа выполняется по вариантам. Для выполнения студентам представлено 6 заданий по работе с различными форматами файлов: xlsx, csv, txt. К работе предлагается краткая инструкция по выполнению. Рекомендуется использовать инструменты библиотеки Pandas.

  3. Лабораторная работа "Парсинг"
    Работа выполняется по вариантам. Каждый вариант содержит 4 задания. К работе предлагается краткая инструкция, содержащая пример парсинга страниц. Рекомендуется использовать для выполнения заданий модули csv, urlopen, BeautifulSoup.

  4. Проектное задание "Diamonds"
    В качестве данных для анализа используется датасет характеристик 54 000 бриллиантов. Данные содержат пропущенные значения. В работе проводится углубленный анализ представленных характеристик: производится расчет среднего значения, медианы, дисперсии, стандартного отклонения. Рекомендуемые библиотеки: Pandas, Matplotlib, NumPy.

  5. Проектное задание "StudPortug"
    В данном проектном задании для обработки представлены данные опроса студентов математического курса средней школы. В работе применяются различные инструменты фильтрации, визуализации. Предлагается использовать методы группировки и агрегирования. Рекомендуемые библиотеки: Pandas, Matplotlib, NumPy, Plotly.

  6. Проектное задание "BabyNames"
    В работе предлагается использовать данные о частоте встречаемости детских имен в США за период с 1880 по 2021 годы. Данные загружаются с сайта Администрации социального обеспечения США в виде архива. Студенты самостоятельно извлекают данные (примерно 2 000 000 строк) и выполняют задания по анализу и визуализации. Рекомендуется использовать модули Requests, Pandas, NumPy, Matplotlib, Plotly, Seaborn.

  7. Итоговое проектное задание
    По завершению курса студентам предлагается выполнить проект, целью которого является анализ рынка игровой индустрии. Из открытых источников доступны данные до 2016 о продажах игр, оценки пользователей и экспертов, жанры и платформы. Студентам предстоит выявить определяющие успешность игры закономерности и спланировать кампанию на 2017 год, сделав ставку на потенциально популярный продукт. В ходе работы студенты решают поставленные перед ними бизнес-задачи, применяя изученные ранее инструменты. Рекомендуется использовать библиотеки Pandas, NumPy, Matplotlib, Plotly, Seaborn, SciPy.
